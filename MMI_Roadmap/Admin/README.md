# Admin

This folder holds MMI's administrative records related to the fuels certification program.

## Contents

| Subfolder / File | Description |
|---|---|
| MC Registration Agreement | Signed agreement with Measurement Canada for ASP status |
| Schedule A | Current list of authorized technicians, device types, and sub-types |
| Client Contracts | Inspection service agreements with fuel retailers and bulk operators |
| Invoices | Billing records for completed inspection services |
| Correspondence | Letters, emails, and notices from Measurement Canada and TSSA |

> Place documents directly in this folder or create subfolders as volume grows.
