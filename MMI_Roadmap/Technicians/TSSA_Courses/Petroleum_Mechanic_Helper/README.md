# Petroleum Mechanic Helper (PMH)

**Authority:** TSSA — Technical Standards and Safety Authority (Ontario)
**Provider:** Fuels Safety On-Line Inc. (fuelssafety.ca)
**Enables:** Replace nozzles, spouts, hoses, breakaways, filters, and belts on pumps and dispensers without supervision. Gateway certification toward full PEM.
