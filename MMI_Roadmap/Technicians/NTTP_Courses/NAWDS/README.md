# Non-Automatic Weighing Devices (NAWDS)

**Authority:** Measurement Canada — National Technical Training Program (NTTP)
**Prerequisite:** NTTP prerequisite exam (separate core course track — can be taken independently of petroleum courses)
**Enables:** Certification of commercial scales and weighing equipment. Significantly widens MMI's ASP scope beyond liquid measurement into the much larger weighing device market.

**Status:** `NOT STARTED`

## Enrollment

- [ ] Confirm prerequisites met (NTTP prerequisite exam)
- [ ] Register for NAWDS course via Measurement Canada NTTP

## Certification Steps

- [ ] Pass theoretical evaluation (written exam — 70% pass mark)
- [ ] Schedule practical (field) evaluation with Measurement Canada
- [ ] Pass practical evaluation
- [ ] Be added to Schedule A for NAWDS device type
